Restart File

Selecting a restart file allows tracking of multiple eruptions. Here is an example from the 2001 eruption of Mt. Cleveland in Alaska.
  • Select Cleveland from the Volcano Name menu.
  • Set 2001 02 19 14:30 in the Eruption Start field.
  • Set eruption hours to 3.
  • Use ncep-reanalysis in the model menu.
  • Click Run Puff
  • Select Cleveland from the Previous menu.
  • Select 2001 02 19 17:30 from the restartFile menu.
  • Set 2001 02 19 17:30 in the Eruption Start field.
  • Select Cleveland from the Volcano Name menu.
  • Click Run Puff
  • Select Cleveland from the Previous menu.
  • Click Map.
That's it!
