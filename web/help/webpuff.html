Getting Started with Webpuff

Quick Start

  • Select a volcano from the scroll list in the upper, left corner. The region of the world will appear in the field below, and the latitude and longitude will appear in the fields on the right.
  • Click "Run Puff"
  • After the page reloads, select the volcano from the "Previous" list in the upper, right corner.
  • After the frame in the lower, left corner loads, click the "Map" button.
  • The first image will appear in the lower, right corner. Use the scroll list in the lower, left frame to view different times.

More Details

Model Options

The gray box in the middle houses the simulation options. Default values exist for all. To get help on a particular option, click on its name. To reset everything to the default, click the "RESET" button.

Mapping Options

The green box in the lower, right corner how the results are mapped. Some default options may exist, such as the map projection. The results from a particular simulation can be mapped repetatively without re-running the eruption simulation. To get help on a particular option, click on its name. Press "Update" to create a new set of maps. Different option sets can be used to view maps of particular interest:
  • Ash fallout Selecting the "fallout" radio button will only map ash on the ground. Select the "isopach" color protocol to get relative thickness predictions.
  • Flight level Set the height range text fields to the minimum and maximum values for a particular flight level. Only ash that exists in that region will be mapped.
