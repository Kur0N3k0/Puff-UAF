latlon

This option explicitly specifies the bounding box of the map. Usually the bounding box is determined to be just big enough to contain all the ash being mapped. The format for this entry is lt1/lt2/ln1/ln2 where
  • lt1: lower (southern) latitude value in degrees north. Range: -90 <= lt1 <= +90.
  • lt2: upper (northern) latitude value in degrees north. Range: -90 <= lt1 <= +90.
  • ln1: left (western) latitude value in degrees east. Range: 0 <= lt1 <= +360.
  • lt2: right (eastern) latitude value in degrees east. Range: 0 <= lt1 <= +360.
Values may be decimal. For example, a map covering the majority of mainland Alaska and some of the Aleutian Islands would be
50/70/190/230
